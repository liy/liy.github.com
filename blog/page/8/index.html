复习数学知识…

Mar 8th, 2008 | Comments

关于 sum-squared error ：E = 1/2*sum ((DATAp-PREDICTIONp)^2)
和
y = mx + b 的 linear regression…
 
仔细想想sum-squared error的公式：平方的意义保持两组数据相互比较以后为正数,只求差距，并不追求两组数据之间的具体关系。
而1/2的意义纯属使制表的时候比较方便。
如果两组数据差距很大的话…平方之后差距会更大…制表会不方便…
平方出来是偶数…乘1/2能尽量缩小差距，方便制图，同时保证为正偶数。
 
反正俺就是这么理解的…
 
至于linear regression….线性回归…我一直在思考这个“回归”到底回归啥了….
难道是通过两组数据Xn，Yn… “回归”出来一个直线方程：y = mx + b ?!?!?
 
所谓“回归”？！？ 如果真是这么个意思….那应该叫 linear anagenesis…. 线性进化…. 数据进化成一个方程了。
哇哈哈哈哈哈….
