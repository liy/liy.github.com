暂时借用一下 Blog 记一下applet的特点

Apr 24th, 2005 | Comments

1. JApplet has a content pane just as a JFrame does.

2. JApplet do not use setVisible method. It do not contain a demonstration main method, it is displayed automatically by Web or an applet viewer.

3. JApplet do not have title…bec web page’s title is difined by HTML, <title>title</title>…

4. It also has no setSize method, and default close operation.

5. Overall, JApplet is embedded in a HTML document.
